==================================
■ プロジェクト概要
==================================
「Pythonでつくるゲーム開発入門講座」に記載されている
ローグライクゲームをベースとして、リファクタリング及び、
ゲームシステムの調整を行いました。

また、オリジナルのコードは約650行弱の単一ファイルでした。

主なリファクタリングの観点とオリジナルからの変更点は、以下の通りです。

・ファイルの分割
・オブジェクト指向の導入
・pylintによる警告の解消

・ゲーム用のシーン管理システム導入
・戦闘システムをコマンド入力タイプからシームレス戦闘に変更

- 参考書籍URL
http://www.sotechsha.jp/pc/html/1239.htm

==================================
■ ファイル概要
==================================
main.py - プログラムの起点
const.py - 定数定義

/event - マップイベント
    event.py - イベント基底クラス
    enemy.py - モンスター
    stairs.py - 階段
    treasure_box.py - 宝箱

/ game - ゲームのコア情報管理
    game_info.py - ゲーム進行情報管理
    game_system.py - ゲームシステム管理

/ image - 画像リソース

/ libs - 汎用ライブラリ
    matrix.py - 正方行列型

/ manager - 各種マネージャー
    event_manager.py - イベント管理
    image_manager.py - 画像管理
    input_manager.py - 入力管理
    sound_manager.py - 音声管理
    text_manager.py - 文字表示管理

/ model - モデル
    dungeon.py - ダンジョン
    item.py - アイテム
    player - プレイヤー

/ scene - シーン
    map_manager.py - マップ管理
    scene.py - シーン基底クラス
    scene_map.py - マップシーン
    scene_title.py - タイトルシーン
    task.py - タスク基底クラス
    task_map_game_over.py - マップ/ゲームオーバー
    task_map_get_item.py - マップ/アイテム取得
    task_map_move.py - マップ/移動
    task_map_scene_to_title.py - マップ/シーンをタイトルに切り替える
    task_map_change_floor.py - マップ/フロアの移動

/ sound - 音声リソース

/ _origin
    one_hour_dungeon.py - リファクタリング前の本記載のコード
